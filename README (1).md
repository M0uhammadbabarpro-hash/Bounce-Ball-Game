# 🎮 Bounce Ball Game

A simple and fun browser-based Bounce Ball Game built using HTML5 Canvas, CSS3, and Vanilla JavaScript.

## Features
- HTML5 Canvas Rendering
- Smooth Ball Physics
- Paddle Controls using Keyboard
- Brick Collision Detection
- Real-Time Score Tracking
- Win & Game Over Screens

## Controls
- Left Arrow: Move Left
- Right Arrow: Move Right

## How to Run
Open `game.html` in any modern web browser.
